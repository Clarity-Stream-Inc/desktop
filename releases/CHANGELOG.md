# Changelog

All notable changes to the Clarity Stream desktop app are documented here.
Format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), versioning follows [SemVer](https://semver.org/).

## [Unreleased]

### Added
- Initial Electron shell wrapping the live `claritystream.io` dashboard
- macOS + Windows installers (signed + notarized)
- Native OS notifications for alerts
- Tray / menu-bar quick-access menu with recent alerts
- Dock / taskbar unread badge count
- Encrypted session storage (Keychain on macOS, DPAPI on Windows)
- Deep-link support for `claritystream://` URLs
- Auto-update via `electron-updater`
